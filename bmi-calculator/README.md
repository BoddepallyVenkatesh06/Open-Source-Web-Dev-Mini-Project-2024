### BMI Calculator
BMI calculator can be used to calculate body mass index of people.

### Use of the Project:
 It can be used in gyms etc to check if a person is overweight or underweight.

### Used Technologies

  * HTML5

  * CSS3

  * JavaScript


#### Steps to Use:

---
- Go to the directory

- Run the index.html file

- Start Calculating!!
