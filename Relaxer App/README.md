<h1>Relaxer App</h1>

<p>Relaxer App written in HTML, CSS, and JavaScript .</p>

### Use of the Project:

<p>This app helps to relax by telling you when to breathe in, hold and breathe out .</p>

## Used Technologies

- HTML5

- CSS3

- JavaScript

### Steps to Use:

---
- Go to the directory

- Run the index.html file

- Just relax!!
